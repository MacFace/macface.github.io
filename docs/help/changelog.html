更新履歴

2006年6月4日 Ver.1.3.0

  • ユニバーサルバイナリ化
  • ディスク消費予測の予測方法および表示方法を改善した
  • 環境設定の顔パターンファイル切り替えメニューで、既に存在していないパターンファイルをグレーアウトするのではなく、そもそも含めないようにした

2005年11月13日 Ver.1.2.2

  • ステータスウインドウ・ディスク消費グラフの表示/非表示を保存するようにした
  • ステータスウインドウのUserやTotalなどの時間が間違っていたのを修正
    (きざみ値の単位を1/60秒から1/100秒にした)
  • ドックのメニューからウインドウを開いた時に最前面にならないのを修正
  • 「ファイル」メニューの「閉じる」、「編集」メニュー全般が使用できない状態であってもグレーアウトされないのを修正
  • ヘルプがMacOS X 10.4で表示されないのを修正

2004年02月03日 Ver.1.2.1

  • ディスク消費グラフの履歴が保存されない不具合を修正
  • MacOS X 10.3で環境設定ウインドウが開けなる不具合を修正

2003年05月23日 Ver.1.2.0

  • 顔パターン切り替え機能を搭載
  • ディスク消費グラフの目盛りの表示がおかしくなる問題を修正

2003年02月11日 Ver.1.1.1

  • MacOS X v10.1.5で動かない不具合を修正
  • ディスク消費グラフの残り日数の計算がおかしかったのを修正
  • ドックメニューのトグル動作をやめ、常に開くようにした
  • ヘルプのディスク消費グラフの Finder との空き容量の食い違いの説明を修正

2003年01月22日 Ver.1.1

  • ディスク消費予測機能を追加
  • グラフ類の表示をちょっとかっこよくした
  • ドックのメニューにもウインドウを開くメニューを追加
  • 表情のメモリ第3段階目に汗マークを付けて、持続時間を修正

2002年12月17日 Ver.1.0.1

  • MacOS X v10.2でヘルプが表示されない不具合を修正

2002年07月11日 Ver.1.0.0

  • 公開
