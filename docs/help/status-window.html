ステータスウインドウ

ステータスウインドウは、CPU とメモリの状態についてより詳しい情報を表示します。上段のメモリの使用状況と、下段の CPU 使用率に別れています。グラフは過去60秒分の状態を表示しています。

上段: メモリの使用状況

グラフの意味は次の通りです。ページイン、ページアウトは縦の棒で表されます。

グラフの色 意味 説明
青 free 未使用のページ
薄いオレンジ inactive 使用中のページのうち、ページアウト候補のもの
オレンジ active 使用中のページ
赤 wired 必ず物理メモリ上に無ければならないページ
白 pagein ディスクから物理メモリへ移動したページの数
黒 pageout 物理メモリからディスクへ移動したページの数

下段: CPU 使用率

グラフは、青い部分がシステムが消費している割合、赤い部分がユーザーが消費している割合を示しています。右側の時間表示はそれぞれが消費した累積の時間です。
